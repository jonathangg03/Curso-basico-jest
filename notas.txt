Truthness / Verdadero

-CORRECCIÓN: Sí tenemos varios describe, y dentro un test diferente, los suite siempre se mostrará un suite, y 
    varios test, porque el suite es el archivo que se esta corriendo.

-Recordar que al usar el .not antes del comparador, ese comparador se volverá negativo.

Comparadores:
    .toBeNull(): Se espera que el elemento del expect sea null.
    .toBeTruthy(): Se espera que el elemento del expect sea true.
    .toBeFalsy(): Se espera que el elemento del expect sea false.
    .toBeUndefined(): Se espera que el elemento del expect sea undefined.
    .not.toBeTruthy(): Se espera que el elemento del expect sea falso ("no verdadero").
