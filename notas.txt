Strings

-En las comparaciones de texto, para ver sí tiene alguna palabra dentro de un string, la palabra se usa dentro de
    "/palabra/".

Comparadores:
    .toMatch(): Sí el string está como paramtro esta dentro del string que estará dentro del string del expect.
    .not.toMatch(): Negativo a la anterior.
    .toHaveLength(): Ver sí el string tiene el tamaño indicado como parametro.
