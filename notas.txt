Common Matchers / Comparadores Comunes

-Test Suites: Cada vez que usamos el describe(); dentro podrá tener varios test().
-Test: Donde se usó test(). Puede haber varias dentro de un suite.

Comparadores comúnes:
    -Usados para comparar 2 elementos.
    -Los mas usados:
        .toBe(): Se esperá que la ejecución de una función que sea parametro del expect, sea igual a lo que
            este dentro de este metodo.
        .toEqual(): Comparación entre lo que esta en el expect() y este metodo, pasará en caso de ser iguales.
            .not.toEqual(): Negación de lo anterior, pasará sí son diferentes.
